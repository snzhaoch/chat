Chat 聊天室
===========================
## 一键部署方式
    ```Bash
    cd configuration
    sudo su
    bash -x setup_development.sh
    ```
------------------------------------------------
## 聊天室简介/演示
    * 登录：
        * 通过输入昵称进入聊天室
        * 无昵称无法登录，重新跳转至首页
    * 聊天室
        * 有大厅、游戏、水区三个聊天室，用户可以自由进出
        * 登录成功后自动进入大厅
        * 用户进入、离开聊天室，聊天室会进行广播
        * 支持多人同时在线聊天
